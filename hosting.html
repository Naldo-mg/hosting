Bienvenido a mi página web

¿Que es un servidor?

Un servidor es una computadora o un sistema de computadoras que proporciona servicios, recursos o datos a otros equipos, llamados "clientes", a través de una red. Puede servir diferentes tipos de recursos, como alojamiento web, almacenamiento de archivos, bases de datos, entre otros.

¿Que es un hosting?

Hosting: El hosting (o alojamiento web) es un servicio que permite a individuos y organizaciones publicar un sitio web en Internet. Este servicio incluye el almacenamiento de los archivos del sitio web en un servidor y su acceso por parte de los usuarios de Internet.

Opciones de servidores y hosting gratuitos:

GitHub Pages

  • Características:
    • Alojamiento gratuito para páginas estáticas.
    • Soporta HTML, CSS y JavaScript.
    • Integración con repositorios de GitHub.
  • Limitaciones:
    • Solo para sitios estáticos (sin back-end).
    • Límite de uso en ancho de banda y almacenamiento.

Netlify

  • Características:
    • Alojamiento para aplicaciones web estáticas y dinámicas.
    • Despliegue continuo y fácil integración con repositorios.
    • Funciones de formularios y funciones de servidor (serverless).
  • Limitaciones:
    • Límite en el uso de ancho de banda (100 GB por mes en el plan gratuito).
    • Algunas funciones avanzadas requieren una suscripción de pago.

Vercel

  • Características:
    • Ideal para proyectos de front-end y aplicaciones JAMstack.
    • Despliegue automático desde repositorios de GitHub, GitLab y Bitbucket.
    • Soporte para funciones serverless.
  • Limitaciones:
    • Límite de uso de ancho de banda (100 GB por mes en el plan gratuito).
    • Puede ser costoso si se exceden los límites.

InfinityFree

  • Características:
    • Alojamiento gratuito con almacenamiento ilimitado.
    • Soporta PHP y MySQL.
    • Sin publicidad forzada.
  • Limitaciones:
    • Límite en el uso de recursos (CPU, RAM, etc.).
    • Soporte limitado comparado con opciones de pago.

FreeHosting

  • Características:
    • Alojamiento gratuito con soporte para PHP y MySQL.
    • Dominio .freehosting.com o uso de dominio propio.
    • Sin límite de transferencia de datos.
  • Limitaciones:
    • Espacio de almacenamiento limitado (1 GB).
    • Soporte técnico limitado.

000webhost

  • Características:
    • Alojamiento gratuito con 1 GB de espacio y 10 GB de transferencia de datos.
    • Soporte para PHP y MySQL.
    • Constructor de sitios web gratuito.
  • Limitaciones:
    • Tiempo de inactividad ocasional en la versión gratuita.
    • Publicidad en versiones gratuitas de algunos planes.
